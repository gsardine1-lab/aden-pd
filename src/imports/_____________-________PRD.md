# 亚丁持仓总览页面 PRD

版本号：V3.2
更新时间：2026-05-15
适用角色：UI 设计师、前端开发工程师
核心目标：一站式、清晰准确地呈现用户通过本平台达成的所有期权持仓状态、盈亏分布及到期信息，提供便捷的行权申请入口与数据导入/导出能力，满足日常持仓管理需求。

---

## 一、页面整体布局

页面采用"顶部标题功能区 → 筛选状态提示条（按需显示） → 核心统计卡片 → 中部功能模块 → 底部明细列表"的五段式垂直布局，最小宽度 1200px。

1. **左侧全局导航栏**：首页、行情、询价、订单、持仓、洞察、日历、提醒、设置。当前页"持仓"高亮。
2. **页面标题与功能按钮区**：左侧页面标题，右侧集中放置数据操作与状态控件。
3. **筛选状态提示条**：任意筛选条件激活时出现，统计卡片跟随筛选结果联动更新。
4. **第一行：核心统计卡片（4 个等宽）**
   - 持仓总市值
   - 持仓预估净收益（昨日）
   - 临近到期（≤7 天）
   - 可行权且盈利
5. **第二行：功能模块（3 列等宽）**
   - 持仓分布（按收益率）—— 饼图 + 图例 + 风险说明
   - 到期日历（未来 30 天）—— 最近 9 个活跃持仓
   - 持仓盈亏情景模拟 —— 可搜索选择器 + 拉杆 + 固定场景
6. **第三行：持仓明细列表**
   - 列表头部：总数统计 + 快捷筛选标签 + 搜索框 + 高级筛选按钮
   - 列表主体：全部持仓明细（金额按原生币种展示，完整数字 + 缩写预览）
   - 列表底部：分页控件


## 二、页面标题与功能按钮区

**区域高度**：48px

**布局**：
- 左侧："持仓总览"（大字体加粗）
- 右侧依次排列：数据导入按钮 → 数据导出按钮 → 最后更新时间（YYYY-MM-DD HH:MM） → 手动刷新按钮

### 数据导入

蓝色主按钮"数据导入 ▾"，点击弹出下拉菜单：
1. 录入外部数据
2. 持仓批量导入
3. 下载持仓模板

### 数据导出

白色次按钮"数据导出 ▾"，点击弹出下拉菜单：
1. 导出当前视图 (Excel)
2. 导出全部持仓 (Excel)
3. 导出为 CSV

两个菜单互斥打开，点击外部关闭。详细交互由对应独立 PRD 定义，本页面仅提供入口。


## 三、筛选体系与联动规则

### 3.1 触发方式（条件可叠加）

| 方式 | 位置 | 说明 |
|------|------|------|
| 快捷筛选标签 | 列表头部同行 | 交易对手、结构、币种、临近到期 |
| 搜索框 | 列表头部右侧 | 聚焦即弹出持仓下拉列表（前8条），输入关键词实时过滤，点击某项填入标的名称并执行搜索 |
| 高级筛选面板 | 点击"筛选"按钮弹出 | 多条件组合筛选 |

### 3.2 快捷筛选标签

平铺在列表头部"共 X 条持仓"右侧，各维度以"|"分隔：

- **交易对手**：`亚丁` `非亚丁`
- **结构**：`100%Call` `103%Call` `105%Call`
- **币种**：`CNY` `USD` `HKD`
- **临近到期≤7天**：红色高亮按钮

点击切换选中/取消，选中时蓝色填充（临近到期为红色）。任意筛选激活时末尾显示蓝色"清空"文字链接。

### 3.3 高级筛选面板

点击"筛选"按钮弹出下拉面板，含以下条件：

| 筛选项 | 控件 | 说明 |
|--------|------|------|
| 结构 | 下拉选择 | 动态提取数据中所有结构值 |
| 状态 | 下拉选择 | 未到期/可行权且盈利/可行权但亏损/已到期/已平仓 |
| 币种 | 下拉选择 | CNY/USD/HKD |
| 交易对手 | 下拉选择 | 亚丁/非亚丁 |
| 期限 | 双日期选择器 | 按到期日期范围筛选 |
| 持仓规模 | 双数字输入 | 名义本金范围（万） |

面板内有"清空全部"按钮，激活筛选时筛选按钮呈蓝色高亮 + 感叹号标记。

### 3.4 联动规则

筛选激活后，顶部 4 个统计卡片均基于筛选结果重新计算展示。筛选状态提示条出现，文案：

> 当前展示 **统计结果** 为已选中 X 个持仓。

"统计结果"四字红色强调。右侧"清空"按钮一键重置所有筛选条件，提示条消失。


## 四、核心统计卡片

### 1. 持仓总市值

- **标题**：持仓总市值
- **主显示**：选中币种的总计市值（大字体加粗），格式 `¥7,609,000（760万+）`
- **副显示**：人民币 / 美元 / 港币分列市值
- **交互**：右上角币种平铺切换按钮（CNY / USD / HKD），选中项白底阴影突出，切换仅影响本卡片；本卡片不支持点击跳转，无 hover 动画

### 2. 持仓预估净收益（昨日）

- **标题**：持仓预估净收益（昨日）
- **主显示**：净收益金额 + 收益率（带正负号），盈红亏绿
- **副显示**：浮盈金额/比例/数量（红）、浮亏金额/比例/数量（绿）
- **数据定义**：整体收益率 = 净收益 / 昨日持仓总市值 × 100%
- **交互**：右上角独立币种平铺切换按钮（CNY / USD / HKD），切换后金额跟随换算，比例和笔数不变；不支持点击跳转，无 hover 动画

### 3. 临近到期（≤7 天）

- **主显示**：临近到期持仓数量（大字体加粗）
- **副显示**："个持仓" + "风险需关注"（红色）
- **交互**：点击弹出持仓列表弹窗，展示全部临近到期持仓的详细信息

### 4. 可行权且盈利

- **主显示**：可行权且盈利持仓数量（大字体加粗）
- **副显示**："个持仓" + "可申请行权"（绿色）
- **交互**：点击弹出持仓列表弹窗，展示全部可行权且盈利持仓的详细信息


## 五、中部功能模块

### 1. 持仓分布（按收益率）

**标题**：持仓分布（按收益率）共 X 个持仓

**布局**：左侧实心饼图 + 右侧图例（浮盈区在上、浮亏区在下） + 底部风险说明

**收益率区间**：以 5% 为步长，从 ≥30% 到 <-30%，共 14 档。正收益区为红色系（深→浅），负收益区为绿色系（浅→深）。仅展示有数据的档位。

**饼图**：
- 实心饼图（无内环），扇区内部白色标签格式 `N (X%)`，始终可见
- 悬停时扇区放大 + 阴影，浮层跟随鼠标移动，显示档位名称、持仓数、占比、总预估净收益
- 点击扇区或图例项弹出持仓列表弹窗，展示该档位全部持仓

**风险说明**：模块底部固定展示香草期权核心特征 —— 亏损有限（最大亏损为期权费）、收益无限、无追保风险、时间价值衰减。末尾附免责声明。

### 2. 到期日历（未来 30 天）

**标题**：到期日历（未来 30 天），右侧排序切换按钮组（"按到期时间" | "按名称"，默认按到期时间）

**展示量**：最近到期的 9 个活跃持仓（不含已平仓和已到期）

**横轴**：当前日期起 30 天，每 7 天一刻度。纵轴：持仓标的。

**每个持仓行**：
- 左侧：标的名称（亚丁持仓前有蓝色圆点标记）+ 可行权标签（红色"行权"或灰色"行权"）+ 代码
- 进度条：红色（盈利）或绿色（亏损），左侧有色条，右侧标注到期日期 MM/DD
- 预估净收益：条内左侧（长条时）或条外右侧（短条时），红/绿颜色跟随盈亏

**底部图例**：
- 蓝色圆点 = 亚丁持仓可申请行权
- 行权标签 = 持仓当前处于可申请行权状态
- 红点 = 临近到期≤7天
- 日期范围：当日 → 30天后

**交互**：悬停高亮，点击该行弹出弹窗显示该持仓详情。行展开时整体微缩放 + 阴影反馈。

### 3. 持仓盈亏情景模拟

**标题**：持仓盈亏情景模拟

**组件**：

1. **可搜索持仓选择器**：输入框内嵌搜索图标，闭合状态显示已选标的（币种符号 + 名称 + 代码），聚焦时清空并展开下拉列表。下拉项格式：`¥ 标的名称 (代码) 到期 YY-MM-DD`。无匹配显示"无匹配结果"，点击外部关闭。

2. **价格调整控件**：-1% 按钮 → 拉杆（-100% ~ +100%） → +1% 按钮。拉杆中心刻度"当前市价 (0%)"。下方实时显示模拟价格与预估盈亏。

3. **固定场景模拟**（与拉杆独立）：
   - 标题"固定场景模拟（与拉杆独立）"明确区分于上方拉杆
   - 纵向列表展示 +10%、+5%、当前市价、-5%、-10% 五个固定点
   - 每行左侧标签、右侧预估盈亏金额，盈上亏下排列
   - 当前市价行蓝色背景高亮
   - 拉杆偏离固定点时，底部新增一行显示拉杆当前位置的结果

**免责声明**：模块底部固定显示"以上模拟结果仅供参考，实际盈亏以行权/平仓时的市场价格和交易规则为准"。


## 六、持仓明细列表

### 6.1 列表头部

单行布局（支持换行）：
- 左侧：`共 X 条持仓`
- 快捷筛选标签（交易对手 → 结构 → 币种 → 临近到期）
- 右侧：搜索框（聚焦弹出下拉列表） + 筛选按钮 + 设置按钮

**设置按钮**：位于筛选按钮右侧。点击弹出下拉面板，列出 6 项可选信息列（当前市价、市价偏离度、盈亏平衡点、盈亏点偏离度、期权费、持仓估值），默认全部勾选。用户取消勾选后对应列隐藏，未全选时设置按钮呈蓝色高亮 + 感叹号标记。选择结果存储至浏览器本地，下次访问时保持。
- 有激活筛选时末尾显示"清空"

### 6.2 排序

点击列表头列标题切换升序/降序，当前排序列标题旁显示 ▲/▼ 蓝色箭头。默认按到期日期升序。

**开仓日/到期日列排序规则**：升序时按开仓日正序排列，降序时按到期日倒序排列。

**全部可排序列**：标的信息、交易对手、结构、操作/状态、开仓日/到期日、持仓名义本金、开仓价、执行价、当前市价、市价偏离度、盈亏平衡点、盈亏点偏离度、期权费、持仓估值、持仓预估净收益、交易规则（按有无特殊规则排序）。

### 6.3 列定义

| 列名 | 内容 | 展示规则 |
|------|------|----------|
| 标的信息 | 标的名称 + 代码 | 名称加粗，代码灰色小字；名称本身为跳转至持仓详情页的链接（ExternalLink 图标） |
| 交易对手 | 亚丁 / 非亚丁 | 亚丁蓝色标签，非亚丁灰色标签；支持列头排序 |
| 结构 | 合约结构（如 100%Call） | 蓝色标签 |
| 操作 / 状态 | 状态标签或行权按钮 | 详见 6.4 操作/状态规则 |
| 开仓日 / 到期日 | 起始日期 / 到期日期 | YYYY-MM-DD；到期≤7天标红加粗显示剩余天数 |
| 持仓名义本金 | 名义本金金额 | 原生币种，完整数字（缩写预览） |
| 开仓价 | 开仓时的标的市价 | 原生币种 |
| 执行价 | 合约执行价格 | 原生币种 |
| 当前市价 | 标的最新市价 | 可选列，默认显示，每 5 分钟自动更新 |
| 市价偏离度 | (市价-执行价)/执行价 | 可选列，盈红亏绿，格式 +X.XX% / -X.XX% |
| 盈亏平衡点 | 执行价 + 单位期权费 | 可选列，原生币种 |
| 盈亏点偏离度 | (市价-平衡点)/平衡点 | 可选列，盈红亏绿，格式 +X.XX% / -X.XX% |
| 期权费 | 总期权费 | 可选列，原生币种，完整数字（缩写预览） |
| 持仓估值 | 当前市场估值 | 可选列，原生币种，完整数字（缩写预览） |
| 持仓预估净收益 | 净收益金额 + 收益率 | 盈红亏绿，收益率带正负号 |
| 交易规则 | 特殊规则标签 | 有特殊规则（敲出/分红调整/非自动行权）显示棕色文字标签，无则显示灰色"标准" |
| 详情 | 详情页入口 | "详情 ↗"链接，表示跳转至独立持仓详情页 |

### 6.4 操作 / 状态规则

| 条件 | 展示 |
|------|------|
| 亚丁 + 可行权且盈利 | 红色"申请行权"按钮 |
| 亚丁 + 可行权但亏损 | 灰色"申请行权"按钮 |
| 亚丁 + 未到期 | 灰色"未到期"标签 |
| 亚丁 + 已到期 | 黄色"已到期"标签 |
| 亚丁 + 已平仓 | 灰色"已平仓"标签 |
| 非亚丁 + 未到期 | 灰色"未到期"标签 + 编辑图标，整体可点击弹出状态编辑弹窗 |
| 非亚丁 + 已平仓 | 灰色"已平仓"标签 + 编辑图标，整体可点击弹出状态编辑弹窗 |
| 非亚丁 + 已到期 | 黄色"已到期"标签（不可编辑） |

> 注：交易对手为"非亚丁"的持仓不支持行权功能，因此不会出现可行权状态。仅亚丁持仓可申请行权。

**编辑状态弹窗**：仅"未到期"和"已平仓"两个选项可选，用户自由切换，点击"确定"关闭。弹窗显示标的名称、代码、交易对手信息。

### 6.5 行展开详情

点击行内任意位置（标的名称为跳转链接除外，操作按钮和详情链接除外）展开详情面板，显示：

- **左侧**：交易规则详情 —— 行权规则、到期行权规则。特殊规则（非自动行权、敲出规则、分红处理）以偏红棕色文字突出显示，与普通规则区分。
- **右侧**：持仓概要 —— 币种、结构、名义本金、期权费。底部附带"查看持仓详情 ↗"链接跳转至独立详情页。

再次点击或点击非亚丁行收起。

### 6.6 分页

每页 12 条，底部分页控件显示页码按钮，当前页蓝色高亮。筛选条件变化时自动回到第 1 页。


## 七、弹窗（DrillDownModal）

### 触发场景

- 点击"临近到期"卡片 → 弹窗标题"临近到期（≤7天）"
- 点击"可行权且盈利"卡片 → 弹窗标题"可行权且盈利"
- 点击持仓分布饼图扇区或图例 → 弹窗标题为对应收益率档位名称
- 点击到期日历持仓行 → 弹窗标题为该标的名称

### 弹窗内容

顶部标题栏（标题 + 持仓总数） + 表格列表 + 底部提示。

表格列：标的信息、交易对手、结构、状态、开仓日/到期日、名义本金、执行价、当前市价、预估净收益、交易规则、详情。

- 状态列中"可申请行权"和"申请行权"为可点击按钮样式（红/灰），其余为静态标签
- "详情"列为跳转链接样式（ExternalLink 图标）
- 所有状态标签和详情链接均设置不换行

弹窗宽度 1100px，最大高度 80vh，超出内容内部滚动。点击背景遮罩关闭。


## 八、全局交互与数据展示规则

### 8.1 金额展示规范

所有金额统一采用"完整数字（缩写预览）"格式：
- 完整数字带千分位分隔符
- 括号内缩写值 + 单位（万/亿），有余数时末尾加"+"表示有余
- 示例：`¥7,609,000（760万+）`、`¥15,000,000（1500万）`
- 不足 1 万仅显示完整数字，不显示缩写

### 8.2 币种规则

- 持仓总市值和持仓预估净收益两个卡片分别拥有独立币种平铺切换按钮（CNY/USD/HKD），互不影响
- 列表中每笔持仓按其开仓时确定的原生币种展示
- 全部金额为实时计算结果

### 8.3 颜色规范

严格遵循国内金融市场标准：红色代表正值/上涨/盈利；绿色代表负值/下跌/亏损。

### 8.4 数据更新

每 5 分钟自动刷新，当前市价同步更新。筛选、排序、分页状态在刷新后保持。

### 8.5 风险提示

- 到期≤7天的持仓：到期日期列标红加粗，到期日历进度条末端有红点标记
- 浮亏>20%的持仓：整行浅绿色背景
- 已到期未行权持仓：整行浅黄色背景 + 醒目标签
