# 亚丁对客端 — 持仓总览模块 PRD

> 版本：v2.0 | 日期：2026-05-25 | 作者：产品团队
> 本文档为持仓总览模块的产品需求规格说明，聚焦业务规则、状态逻辑与数据约束，不包含 UI 视觉规范。

---

## 一、产品概述

### 1.1 模块定位

持仓总览是亚丁对客端的核心模块，为用户提供名下期权持仓的全生命周期管理，包括持仓查看、筛选分析、盈亏模拟、外部数据录入、批量导入及历史回溯。

### 1.2 核心能力

- 多维度筛选与搜索，快速定位目标持仓
- 关键事件时间轴，完整追溯每笔持仓的历史变化
- 情景模拟器，预估不同标的价格变动下的盈亏
- 外部持仓录入与批量导入，支持非系统内建仓数据的纳管
- 历史持仓汇总，统计已实现损益

### 1.3 适用范围

当前版本仅支持 **Call 香草看涨期权**。所有字段定义、计算逻辑、规则展示均以此为基准。

## 二、核心业务规则

### 2.1 到期行权规则

| 交易对手 | 到期行为 | 说明 |
|----------|----------|------|
| 亚丁 | 到期自动行权，不论盈亏 | 到期即进入历史持仓，不保留 expired 状态 |
| 非亚丁 | 到期后保留 expired 状态 | 需用户手动平仓后才进入历史持仓 |

### 2.2 行权规则

- 亚丁持仓到达最早行权日后，进入可行权状态
- 可行权且盈利：红色强调，提示用户可申请行权
- 可行权但亏损：红色强调，仍可申请行权
- 申请行权截止时间：交易日下午 13:50
- 行权结算价：最后一小时 TWAP

### 2.3 敲出规则

- **强制敲出**：标的连续 3 日涨停或跌停时触发
- **协商敲出**：双方协商一致后执行

### 2.4 分红规则

| 规则类型 | 说明 |
|----------|------|
| 调整 | 分红后执行价相应调整 |
| 不调整 | 分红不影响执行价 |
| 扣分红 | 提前行权需扣除分红金额 |

### 2.5 交易对手差异总表

| 维度 | 亚丁 | 非亚丁 |
|------|------|--------|
| 到期处理 | 自动行权 → closed | 保留 expired → 手动平仓 |
| 主列表展示 | expired 不展示 | expired 展示（可操作） |
| 操作按钮 | 可行权时"申请行权" | 未到期/已到期时"手动平仓" |
| 交易规则展示 | 展示完整 4 项规则 | 不展示 |
| 关键事件 | 6 种全量事件 | 4 种（无除权除息、分红调整） |
| 建仓/行权订单号 | 有 orderId | 无 orderId |
| 字段编辑 | 只读 | 可编辑 |

## 三、持仓状态定义与流转

### 3.1 五种状态

| 状态 | 英文标识 | 说明 | 适用对手 |
|------|----------|------|----------|
| 未到可行权日 | not-expired | 尚未到达最早行权日，不可行权 | 全部 |
| 可行权且盈利 | profitable-exercisable | 已到行权日且当前价高于执行价 | 仅亚丁 |
| 可行权但亏损 | loss-exercisable | 已到行权日但当前价不高于执行价 | 仅亚丁 |
| 已到期 | expired | 已过到期日，等待手动平仓 | 仅非亚丁 |
| 已平仓 | closed | 最终态，进入历史持仓 | 全部 |

### 3.2 状态流转

**亚丁持仓**：
```
未到可行权日 → 可行权且盈利 → 已平仓
              → 可行权但亏损 → 已平仓
```

**非亚丁持仓**：
```
未到可行权日 → 已到期 → 已平仓（手动）
```

### 3.3 部分平仓

- 非亚丁持仓支持多次部分平仓
- 每次平仓扣减对应名本
- 当剩余名本 ≤ 0 时，状态自动转为已平仓
- 平仓记录持久化存储于 localStorage

## 四、数据模型

### 4.1 持仓核心字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 标的名称 | 文本 | 标的证券名称，如"贵州茅台" |
| 标的代码 | 文本 | 证券代码，如"600519.SH" |
| 币种 | CNY/USD/HKD | 标的计价币种 |
| 结构 | 文本 | 如 100%、103%、105% 等 |
| 期限 | 文本 | 如"6个月"、"2周" |
| 交易对手 | 文本 | 简称，如"亚丁"、"银河证券" |
| 开仓日 | 日期 | YYYY-MM-DD |
| 到期日 | 日期 | YYYY-MM-DD |
| 最早行权日 | 日期(可选) | 亚丁持仓有此字段 |
| 开仓名本 | 数字 | 建仓时的名义本金(元)，恒定不变 |
| 持仓名本 | 数字 | 当前名义本金(元)，部分平仓后会减少 |
| 开仓价 | 数字 | 建仓时的标的价格 |
| 执行价 | 数字 | 开仓价 × 结构比例 |
| 当前市价 | 数字 | 最新行情价格 |
| 盈亏平衡点 | 数字 | 执行价 + (期权费 / 名本 × 执行价) |
| 期权费 | 数字 | 期权权利金(元) |
| 期权费率 | 百分比 | 期权费占名本的比例 |
| 持仓估值 | 数字 | 理论估值(元) |
| 预估净收益 | 数字 | 理论盈亏金额(元) |
| 收益率 | 小数 | 预估净收益 / 期权费 |
| 数据来源 | system/external | 系统建仓或外部录入 |

### 4.2 平仓记录

| 字段 | 说明 |
|------|------|
| 平仓日期 | 手动平仓的日期 |
| 平仓价格 | 当时的标的价格 |
| 本次平仓名本 | 本次平仓的名本(万) |
| 累计名本 | 该持仓截止当前的已平仓名本总和 |

### 4.3 关键事件类型

| 序号 | 事件类型 | 适用持仓 | 说明 |
|------|----------|----------|------|
| 1 | 建仓 | 全部 | 含开仓价、成交名本、订单号(仅亚丁) |
| 2 | 停复牌 | 有行情数据 | 标的停牌或复牌记录 |
| 3 | ST 记录 | 有行情数据 | 标的被 ST 或摘帽 |
| 4 | 除权除息 | 仅亚丁 | 除权除息日期及比例 |
| 5 | 分红调整 | 仅亚丁 | 分红金额及调整方式 |
| 6 | 行权/平仓结算 | 全部 | 含行权价格、结算金额、结算状态 |

## 五、持仓总览（主仪表盘）

### 5.1 核心统计卡片

| 卡片 | 计算逻辑 | 交互 |
|------|----------|------|
| 持仓总市值 | 全部未平仓持仓的 valuationCNY 之和，支持按币种查看 | 币种切换 |
| 持仓预估净收益 | 全部未平仓持仓的 pnlCNY 之和 + 整体收益率；拆分为浮盈/浮亏数量 | 币种切换 |
| 临近到期（≤7天） | 到期日在未来 7 天内的持仓数 | 点击直接筛选 |
| 可行权且盈利 | status 为 profitable-exercisable 的持仓数 | 点击直接筛选 |

### 5.2 收益率分布图

- 将持仓按收益率分为 14 个桶（从 < -30% 到 ≥ 30%，每 5% 一档）
- 盈利用红色系，亏损用绿色系
- 点击扇区或图例直接筛选对应收益率区间的持仓

### 5.3 近 30 日到期日历

- 左侧日期列表，标注每天到期合约数（≥3 红色，<3 灰色）
- 右侧展示选中日期的持仓卡片
- 卡片含：标的、行权状态、PnL、对手方、结构、期限
- 点击卡片跳转持仓详情页

### 5.4 持仓列表

- 展示所有非 closed 且非(expired+亚丁)的持仓
- 支持排序、分页（每页 12 条）
- 可选列的显示/隐藏持久化到 localStorage
- 到期行黄色底色高亮，重度亏损（< -20%）浅绿底色
- 亚丁持仓显示交易规则标签（敲出规则、分红规则）

## 七、持仓详情页

### 7.1 视图分流

根据持仓状态自动切换：

- **未平仓视图**：关键事件记录 + 持仓明细 + 情景模拟器 + 操作按钮
- **已平仓/已到期视图**：平仓净收益汇总 + 平仓记录表 + 只读持仓概要

### 7.2 关键事件记录

- 时间轴 + 股价走势图，事件标记在对应时间点
- 6 种事件类型，默认展开第一条
- 点击图例切换查看不同事件详情
- 外部录入持仓（无行情数据）：仅显示建仓和行权事件

### 7.3 持仓明细

| 信息组 | 字段 |
|--------|------|
| 时间信息 | 开仓日、到期日、最早行权日、剩余自然日、剩余交易日 |
| 名本与数量 | 开仓名本、持仓名本、期权费、期权费率 |
| 价格信息 | 开仓价、执行价、当前市价、当前涨幅、盈亏平衡点、距平衡点 |

### 7.4 情景模拟器

- 标的价格变动范围：-100% ~ +100%
- 固定场景：+10%、+5%、当前(0%)、-5%、-10%
- 滑块自由调节，实时计算预估盈亏
- 计算公式：当价格 ≤ 执行价时，亏损全部期权费；价格 > 执行价时，收益 = (价格 - 执行价) × (名本 / 执行价) - 期权费

### 7.5 交易规则（仅亚丁）

展示 4 项规则卡片：行权规则、到期行权规则、敲出规则、分红规则

### 7.6 标签与备注

- 支持自定义标签，标签库跨持仓共享
- 支持备注编辑
- 亚丁持仓只读，非亚丁可编辑

## 八、录入外部持仓

### 8.1 布局

左栏文本解析区 + 右栏确认板，双栏并排。

### 8.2 文本解析

- 粘贴期权确认书或交易流水文本
- 正则匹配提取：标的名称、标的代码、结构、期限、交易对手、开仓日、到期日、开仓名本、开仓价、期权费率
- 解析结果为非覆盖式填充（仅更新匹配到的字段）

### 8.3 字段联动规则

| 触发字段 | 联动字段 | 计算方式 |
|----------|----------|----------|
| 标的名称 | 标的代码 | 名称-代码映射表查询 |
| 标的代码 | 标的名称、币种 | 代码-名称映射 + 币种判定 |
| 期限 | 开仓日、到期日 | 期限数字 + 单位(周/月)计算日期 |
| 开仓名本 + 期权费率 | 期权费 | 名本(万) × 10000 × 费率% ÷ 100 |
| 开仓名本 + 开仓价 | 期权数量 | 名本(万) × 10000 ÷ 开仓价 |
| 开仓价 + 结构 | 执行价 | 开仓价 × 结构百分比 ÷ 100 |

### 8.4 币种判定规则

- 已知映射表查询优先
- A 股（代码含 .SH/.SZ 或纯 6 位数字）：CNY
- 港股（HSI）：HKD
- 美股（SPX）：USD
- 默认：CNY

### 8.5 交易对手

- 输入框独立，支持历史下拉（最近 5 条）
- 历史记录持久化到 localStorage
- 提交时自动将新对手方加入历史

### 8.6 校验与提交

- 13 个必填字段全部填写才可提交
- 必填项为空时按钮 disabled，点击闪烁提示缺失字段

## 九、批量导入

### 9.1 导入流程

上传文件 → 自动解析 → 数据审核(可编辑) → 聚合看板校验 → 确认导入

### 9.2 支持格式

.xlsx / .xls / .csv

### 9.3 校验规则（极简）

- **阻断（error）**：必填字段为空 → 该行不可导入
- **提醒（warning）**：标的名称或代码不在已知映射中 → 可导入但后续展示可能缺失数据
- **正常（ok）**：无问题

注意：不校验日期合理性、费率量级、名本量级、结构有效性、周末等。用户填什么就是什么，不替用户判断业务合理性。

### 9.4 聚合看板

- **总览**：总名本、总期权费、总持仓数、涉及机构数
- **按交易对手拆分**：每家对手的笔数、名本合计、期权费合计，按名本降序
- **交互**：点击对手名称直接筛选列表，再次点击取消筛选

### 9.5 编辑与联动

- 数据审核表格中，点击任意单元格即可编辑
- 编辑后自动触发字段联动计算（规则同 8.3）
- 编辑后自动重新校验，状态实时刷新

### 9.6 确认导入

- 跳过所有 error 行，仅导入 ok 和 warning 行
- 写入 localStorage externalPositions
- 导入完成后跳转回持仓总览

### 10.3 列表

- 支持多列排序
- 搜索标的名称/代码

## 十一、金额展示规范

### 11.1 单位与格式

- 名本统一用"万"为单位 + 币种后缀，如"500万 CNY"
- 收益类金额展示精确值，不简写
- 币种简写：CNY、USD、HKD（不使用 ¥、$、HK$）

### 11.2 币种换算

- 内部存储统一为 CNY
- 展示时按汇率换算：USD = 7.23, HKD = 0.927
- 用户可切换币种查看统计数据

### 11.3 正负号与颜色

- 正值：红色显示，"+"前缀
- 负值：绿色显示，"-"前缀
- 零值：灰色显示

## 十四、计算规则汇总

### 14.1 字段联动

| 公式 | 说明 |
|------|------|
| 执行价 = 开仓价 × 结构% | 结构为百分比数值 |
| 期权费 = 名本(万) × 10000 × 费率% ÷ 100 | — |
| 期权数量 = 名本(万) × 10000 ÷ 开仓价 | — |
| 盈亏平衡点 = 执行价 + (期权费 ÷ 名本 × 执行价) | — |
| 距平衡点 = (当前价 - 盈亏平衡点) ÷ 盈亏平衡点 | — |

### 14.2 情景模拟 PnL

- 价格 ≤ 执行价：PnL = -期权费（全部亏损）
- 价格 > 执行价：PnL = (价格 - 执行价) × (名本 / 执行价) - 期权费

### 14.3 收益率

- 收益率 = 预估净收益 ÷ 期权费
- 已平仓：收益率 = 平仓净收益 ÷ 期权费

## 十五、白箱测试要点

### 15.1 状态机流转

- 亚丁：not-expired → profitable-exercisable/loss-exercisable → closed，无 expired 态
- 非亚丁：not-expired → expired → closed（手动平仓），无 exercisable 态
- 部分平仓：剩余名本 ≤ 0 时自动 closed
- 主列表过滤：closed 和 (expired + 亚丁) 均不展示

### 15.2 筛选逻辑

- 各筛选项 AND 逻辑叠加
- "非亚丁"筛选排除亚丁
- 标签多选，任一匹配即通过
- 收益率区间与净收益区间可叠加使用

### 15.3 关键事件生成

- 外部持仓（source=external）：仅建仓 + 行权
- 有行情数据：所有对手方 4 种基础事件 + 亚丁额外 2 种（除权除息、分红调整）
- 行权事件：有 exerciseStatus 字段（预估/已结算/未成交）

### 15.4 批量导入校验

- 必填为空 → error（最小校验原则）
- 标的未识别 → warning
- 不进行业务合理性校验
- 编辑后自动重新校验

### 15.5 金额展示

- 盈红亏绿，正负号区分
- 名本"万"单位
- 币种简写后缀
- 内部统一 CNY 存储

### 15.6 亚丁 vs 非亚丁差异

- 状态机流转路径不同
- 操作按钮文案和行为不同
- 交易规则展示不同
- 关键事件种类不同
- 字段可编辑性不同
- 建仓/行权订单号有无
- 到期处理逻辑不同
